Iconos CSS

Los iconos se pueden agregar fácilmente a su página HTML mediante el uso de una
biblioteca de iconos.

Cómo agregar iconos

La forma más sencilla de agregar un icono a su página HTML es con una biblioteca de
icono, como Font Awesome.

Agregue el nombre de la clase de icono especificada a cualquier elemento HTML en línea
(como <i> o <span>

Todos los iconos en la bibliotecas de iconos a continuación son vectores escalables que
se pueden personalizazzr con CSS (tamaño, color, sombre, etc.)

Font Awesome Icons

Para usar los iconos de Font Awesome, vaya a fontawesome.com, inicie sesión y obtenga
un código para agregar en la <head> sección de su página HTML:

<script src="https://kit.fontawesome.com/yourcode.js" crossorigin="anonymous"><script>

Ejemplo:

<i class="fas fa-cloud"></i>
<i class="fas fa-heart"></i>
<i class="fas fa-car"></i>
<i class="fas fa-file"></i>
<i class="fas fa-bars"></i>

Bootstrap Icons

Para usar los glyphicons de Bootstrap, agregue la siguiente línea dentro de la
<head> sección de su página HTML:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">

Ejemplo:

<i class="glyphicon glyphicon-cloud"></i>
<i class="glyphicon glyphicon-remove"></i>
<i class="glyphicon glyphicon-user"></i>
<i class="glyphicon glyphicon-envelope"></i>
<i class="glyphicon glyphicon-thumbs-up"></i>
