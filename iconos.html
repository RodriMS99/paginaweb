Cómo usar los iconos en HTML

Es muy fácil incorporar iconos a tu página web. Este es un ejemplo pequeño:

<span class="material-icons">face</span>

En este caso se pintaría una cara.

En este ejemplo, se usa una función tipográfica llamada ligaduras, que permite renderizar un glifo de icono mediante su nombre textual.

Para los nombre que no admiten ligaduras, recurre a la especificación de icnos mediante referencias numéricas de caracteres, como en el siguiente ejemplo:

face<span class="material-icons">#xE87C;</span>

Cómo dar estilos a los iconos en Material Desing

Estos iconos se diseñaron para cumplir con los lineamientos de material desing, y se ven mejor cuando se usan los tamaños y colores de iconos recomendados
Los siguientes estilos facilitan la aplicación de nuestros tamaños, colores y estados de actividad recomendados.

/* Rules for sizing the icon. */ .material-icons.md-18 { font-size: 18px; }
.material-icons.md-24 { font-size: 24px; }
.material-icons.md-36 { font-size: 36px; }
.material-icons.md-48 { font-size: 48px; }
/* Rules for using icons as black on a light background. */ .material-icons.md-dark { color: rgba(0, 0, 0, 0.54); }
.material-icons.md-dark.md-inactive { color: rgba(0, 0, 0, 0.26); }
/* Rules for using icons as white on a dark background. */ .material-icons.md-light { color: rgba(255, 255, 255, 1); }
.material-icons.md-light.md-inactive { color: rgba(255, 255, 255, 0.3); }

Dimensionamiento

Aunque los iconos de la fuente se pueden ajustar a cualquier tamaño, de acuerdo con los
lineamientos de icnos de material desing, se recomienda que se muestren en 18, 24, 36 o 48
píxeles. El tamaño predeterminado es de 24 px.

Reglas de CSS para los lineamientos de tamaño estándar de Material Desing:

.material-icons.md-18 { font-size: 18px; }
.material-icons.md-24 { font-size: 24px; }
.material-icons.md-36 { font-size: 36px; }
.material-icons.md-48 { font-size: 48px; }

Los iconos de material se ven mejor en 24 px, pero si un icono debe mostrarse en un tamaño
alternativo, usar las reglas de CSS anteriores puede ser útil:

face<span class="material-icons md-18">face</span>

face<span class="material-icons md-24">face</span>

face<span class="material-icons md-36">face</span>

face<span class="material-icons md-48">face</span>

Colorear

La fuente del icono permite darle estilo fácilmente a un icono de cualquier color.
Se recomienda utilizar iconos negros con una opacidad del 54% o blancos con una opacidad del 100%.

Ejemplos en los que se usan los estilos de CSS de material descritos con anterioridad:

.material-icons.md-dark { color: rgba(0, 0, 0, 0.54); }
.material-icons.md-dark.md-inactive { color: rgba(0, 0, 0, 0.26); }

.material-icons.md-light { color: rgba(255, 255, 255, 1); }
.material-icons.md-light.md-inactive { color: rgba(255, 255, 255, 0.3); }

Ejemplo para dibujar un icono sobre un fondo claro con un color oscuro de primer plano:

face<span class="material-icons md-dark">face</span>

Normal

face<span class="material-icons md-dark md-inactive">face</span>

Inhabilitada

Ejemplo para dibujar un icono sobre un fondo oscuro con un color de primer plano claro:

face<span class="material-icons md-dark">face</span>

Normal

face<span class="material-icons md-dark md-inactive">face</span>

Inhabilitada

Para establecer un color de icono personalizado, define una regla de CSS que especifique el color
deseado para la fuente:

.material-icons.orange600 { color: #FB8C00; }

Ejemplo:

Iconos de imágenes para la Web

Los iconos de material también están disponibles como imágenes normales, tanto en formato
PNG como SVG.
