Tema 1. Planificación de interfaces gráficas

En este capitulo se presentan los fundamentos del diseño de interfaces mediante tecnologias web haciendo especial hincapie en el lenguaje HTML5, además de conocer distintas herramientas para generar interfaces web.

Tema 2. Uso de estilos

Uso de estilos: Las hojas de estilo nos permiten personalizar una web para que su contenido se adapte a las pantallas de los diferentes dispositivos de forma transparente para el usuario, que tendrá la sensación de estar navegando siempre por el mismo sitio independientemente de que acceda con su teléfono móvil, tablet o PC

Tema 3. Contenido multimedia

El contenido multimedia hace referencia a objetos o sistemas que reúnen distintos medios de expresión y presentación de la información, como texto, imágenes, animaciones, video o audios. Los diferentes formatos multimedia digital buscan mejorar la experiencia del usuario.
